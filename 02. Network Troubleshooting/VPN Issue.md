# 🔐 VPN Troubleshooting

## Issue

User cannot connect to company VPN.

---

## Possible Causes

- Internet connection issue
- Incorrect credentials
- MFA failure
- VPN client problem
- Firewall restrictions

---

# Troubleshooting Steps

## 1. Check Internet Connection

Confirm normal internet access.

---

## 2. Verify Credentials

Check:

- Username
- Password
- MFA approval

---

## 3. Restart VPN Client

Close and reopen VPN application.

---

## 4. Check VPN Settings

Verify:

- Correct VPN server
- Correct connection profile

---

## 5. Check Logs

Review:

Event Viewer

→ Windows Logs


---

# Resolution

| Problem | Solution |
|---|---|
| Login failure | Verify credentials |
| MFA issue | Reset MFA |
| Client issue | Reinstall VPN software |
| Network block | Check firewall |
