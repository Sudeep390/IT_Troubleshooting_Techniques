# 📶 WiFi Troubleshooting

## Issue

User cannot connect to wireless network.

---

## Possible Causes

- Incorrect password
- Weak signal
- WiFi adapter disabled
- Driver issue
- Router problem

---

# Troubleshooting Steps

## 1. Check WiFi Status

Verify:

- WiFi enabled
- Airplane mode disabled
- Correct network selected

---

## 2. Forget and Reconnect Network

Path:

Settings

→ Network & Internet

→ WiFi

→ Manage Known Networks


---

## 3. Restart WiFi Adapter

Disable and enable adapter.

---

## 4. Update Driver

Open:
Device Manager

→ Network Adapters

Update driver.

---

## 5. Test Another Device

Check if issue affects:

- Only one device
- Entire network

---

# Resolution

| Problem | Solution |
|---|---|
| Wrong password | Re-enter credentials |
| Weak signal | Move closer |
| Driver issue | Update driver |
| Router problem | Restart router |
