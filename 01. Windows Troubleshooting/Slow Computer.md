# 🐌 Slow Computer Troubleshooting

## Issue

User reports computer is running slowly, applications freeze, or system performance is poor.

---

## 🔍 Possible Causes

- Too many startup applications
- High CPU/RAM usage
- Low disk space
- Malware infection
- Outdated drivers
- Failing HDD/SSD
- Insufficient RAM

---

## 🛠 Troubleshooting Steps

### 1. Gather Information

Ask:

- When did the issue start?
- Any recent changes?
- Is the whole system slow or one application?

---

### 2. Check System Performance

Open Task Manager:
Ctrl + Shift + Esc
Check:

- CPU usage
- Memory usage
- Disk usage
- Running processes

---

### 3. Check Startup Programs

Path:
Task Manager
→ Startup Apps
Disable unnecessary applications.

---

### 4. Check Storage

Verify free space:
File Explorer
→ This PC
→ C: Drive
Keep at least 15-20% free space.

---

### 5. Run System Checks

Check Windows files:
Check disk:
System information:
---

## ✅ Solutions

| Problem | Fix |
|-|-|
| High CPU usage | Close unnecessary apps |
| Low storage | Delete temporary files |
| Too many startups | Disable startup apps |
| Malware | Run antivirus scan |
| Old HDD | Upgrade to SSD |
| Low RAM | Increase memory |

---

## 📝 Service Desk Ticket Example

**Issue:**  
User reports slow laptop performance.

**Diagnosis:**  
High memory usage caused by multiple background applications.

**Resolution:**  
Closed unnecessary applications and disabled startup programs.

**Status:**  
Resolved ✅
