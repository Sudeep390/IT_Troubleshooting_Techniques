# 🔑 Password Reset

## Issue

User cannot log in because they have forgotten their password or it has expired.

---

## Possible Causes

- Forgotten password
- Password expired
- Password does not meet company policy

---

## Troubleshooting Steps

### 1. Verify User Identity

Confirm the user's identity according to company security procedures.

### 2. Open Active Directory Users and Computers (ADUC)

Locate the user account.

### 3. Reset Password

Right-click the user account.

```
Reset Password
```

- Enter a temporary password
- Select **User must change password at next logon** (if required)

### 4. Ask User to Sign In

Confirm the user can log in successfully.

---

## Resolution

| Problem | Solution |
|---------|----------|
| Forgotten password | Reset password |
| Password expired | Assign temporary password |
