# 👤 User Creation

## Issue

A new employee requires access to company systems.

---

## Troubleshooting Steps

### 1. Open ADUC

```
Active Directory Users and Computers
```

### 2. Create New User

```
Users
→ New
→ User
```

Complete:

- First Name
- Last Name
- Username
- Temporary Password

### 3. Configure Account

Assign:

- Department
- Job Title
- Manager (if applicable)

### 4. Add Security Groups

Examples:

- IT
- HR
- Finance
- Sales

### 5. Verify Access

Confirm the user can:

- Log in
- Access email
- Access shared folders
- Access required applications

---

## Best Practice

Follow the **Principle of Least Privilege**, granting only the permissions required for the user's role.
