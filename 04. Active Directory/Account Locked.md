
# 🔒 Account Locked

## Issue

User receives **"Your account has been locked"** and cannot log in.

---

## Possible Causes

- Multiple failed login attempts
- Cached old password
- Password recently changed

---

## Troubleshooting Steps

### 1. Open ADUC

Locate the user account.

### 2. Unlock Account

```
Properties
→ Account
→ Unlock Account
```

### 3. Verify Credentials

Confirm the user is using the correct password.

### 4. Check Saved Credentials

Remove outdated passwords from Windows Credential Manager if necessary.

---

## Resolution

| Problem | Solution |
|---------|----------|
| Failed login attempts | Unlock account |
| Cached credentials | Update saved password |
