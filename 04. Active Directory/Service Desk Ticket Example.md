# 🎫 Service Desk Ticket Example

## Issue

User unable to log in.

## Investigation

Checked Active Directory and found the account was locked after multiple failed login attempts.

## Root Cause

Incorrect password entered repeatedly.

## Resolution

Unlocked the account and confirmed successful login.
