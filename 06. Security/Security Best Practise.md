# 🔒 Security Best Practices

## Objective

Follow security best practices to protect company systems and data.

---

## Password Security

- Use strong passwords
- Do not reuse passwords
- Change passwords when required
- Never share passwords

---

## Multi-Factor Authentication (MFA)

Always enable MFA whenever available.

Examples:

- Microsoft Authenticator
- SMS Verification
- Security Key

---

## Email Security

Always:

- Verify sender
- Check links before clicking
- Avoid unknown attachments
- Report phishing emails

---

## Device Security

- Lock computer when away
- Install Windows updates
- Keep antivirus enabled
- Encrypt company devices

Shortcut:

```
Windows + L
```

Lock your computer immediately.

---

## Internet Safety

Avoid:

- Unknown websites
- Pirated software
- Suspicious downloads

Only install approved software.

---

## Data Protection

- Store company files securely
- Do not use personal USB devices
- Follow company data handling policies
- Back up important data

---

## Physical Security

- Lock office when leaving
- Wear ID badge
- Do not allow tailgating
- Keep confidential documents secure
