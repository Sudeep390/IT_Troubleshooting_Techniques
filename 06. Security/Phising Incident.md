# 🎣 Phishing Incident

## Issue

User receives a suspicious email asking for passwords, payment details, or sensitive information.

---

## Symptoms

- Unexpected email
- Suspicious sender
- Poor grammar
- Urgent requests
- Unexpected attachments
- Unknown links

---

## Possible Causes

- Phishing attack
- Business Email Compromise (BEC)
- Fake login page
- Malicious attachment

---

## Troubleshooting Steps

### 1. Do Not Click Anything

Do not open:

- Links
- Attachments

---

### 2. Verify the Sender

Check:

- Email address
- Domain name
- Spelling

---

### 3. Report the Email

Report to:

- IT Service Desk
- Security Team

Delete the email after reporting.

---

### 4. If User Clicked the Link

Immediately:

- Disconnect from the network
- Change password
- Reset MFA (if required)
- Run antivirus scan

---

## Resolution

| Problem | Solution |
|---------|----------|
| Suspicious email | Report and delete |
| Link clicked | Change password |
| Credentials exposed | Reset password & MFA |
