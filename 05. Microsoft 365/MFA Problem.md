# 🔐 Multi-Factor Authentication (MFA) Problem

## Issue

User cannot complete MFA verification.

---

## Possible Causes

- Incorrect phone number
- Authenticator app issue
- Time sync issue
- Lost mobile device
- User not enrolled

---

## Troubleshooting Steps

### 1. Verify User Identity

Follow company verification policy.

---

### 2. Check MFA Method

Verify:

- Microsoft Authenticator
- SMS
- Phone call

---

### 3. Reset MFA

Using Microsoft 365 Admin Center.

---

### 4. Re-register MFA

Guide the user through setting up Microsoft Authenticator again.

---

## Solutions

| Problem | Fix |
|---------|-----|
| Lost phone | Reset MFA |
| Authenticator issue | Re-register device |
| Wrong phone number | Update contact details |
| Verification failed | Reset MFA methods |
