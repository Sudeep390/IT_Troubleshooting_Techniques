# 📧 Outlook Problems

## Issue

User cannot send or receive emails, Outlook is slow, or Outlook won't open.

---

## Possible Causes

- No internet connection
- Outlook profile corruption
- Mailbox full
- Microsoft 365 service outage
- Authentication issue

---

## Troubleshooting Steps

### 1. Check Internet Connection

Verify the user has internet access.

---

### 2. Restart Outlook

Close Outlook completely and reopen it.

---

### 3. Check Mailbox Storage

Verify mailbox has available space.

---

### 4. Create a New Outlook Profile

Path:

```
Control Panel
→ Mail
→ Show Profiles
→ Add
```

---

### 5. Repair Outlook

```
Control Panel
→ Programs
→ Microsoft 365
→ Change
→ Quick Repair
```

If required:

```
Online Repair
```

---

## Solutions

| Problem | Fix |
|---------|-----|
| Outlook won't open | Create new profile |
| Cannot send email | Check internet & mailbox |
| Slow Outlook | Repair Microsoft 365 |
| Authentication error | Sign out and sign in |
