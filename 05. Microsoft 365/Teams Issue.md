# 👥 Microsoft Teams Issues

## Issue

User cannot join meetings, sign in, or use Teams properly.

---

## Possible Causes

- Internet issue
- Teams cache corruption
- Outdated Teams application
- Microsoft 365 authentication issue
- Service outage

---

## Troubleshooting Steps

### 1. Check Internet

Verify connection.

---

### 2. Restart Teams

Completely close Teams.

Reopen application.

---

### 3. Clear Teams Cache

Close Teams.

Delete Teams cache files.

Restart Teams.

---

### 4. Check Updates

```
Teams
→ Settings
→ Check for Updates
```

---

### 5. Verify Microsoft 365 Login

Sign out and sign back in.

---

## Solutions

| Problem | Fix |
|---------|-----|
| Teams won't open | Restart application |
| Cannot join meeting | Verify internet |
| Login issue | Reauthenticate account |
| Cache problem | Clear Teams cache |
