# ☁️ OneDrive Sync Issue

## Issue

Files are not syncing between the computer and OneDrive.

---

## Possible Causes

- No internet
- Storage full
- Sync paused
- User signed out
- File conflict

---

## Troubleshooting Steps

### 1. Check Internet

Verify connection.

---

### 2. Verify OneDrive Status

Check OneDrive cloud icon.

---

### 3. Resume Sync

If paused:

```
Resume Sync
```

---

### 4. Sign Out & Back In

Reconnect Microsoft account.

---

### 5. Restart OneDrive

Close OneDrive.

Open again.

---

## Solutions

| Problem | Fix |
|---------|-----|
| Sync paused | Resume syncing |
| Storage full | Free storage |
| Login issue | Sign in again |
| Sync error | Restart OneDrive |
