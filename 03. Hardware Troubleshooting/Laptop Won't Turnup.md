# 🔌 Laptop Won't Turn On

## Issue

User reports the laptop does not power on or shows no display.

## Possible Causes

- Battery drained
- Faulty charger
- Loose power cable
- RAM issue
- Hardware failure
- Motherboard issue

---

## Troubleshooting Steps

### 1. Check Power

Verify:

- Charger connected properly
- Power outlet working
- Charging LED is on

---

### 2. Perform a Hard Reset

- Disconnect charger
- Hold Power button for 15–20 seconds
- Reconnect charger
- Power on the laptop

---

### 3. Remove External Devices

Disconnect:

- USB devices
- External monitors
- Docking stations

Restart the laptop.

---

### 4. Check Display

Confirm:

- Screen brightness
- External monitor connection
- Display output (Windows + P)

---

### 5. Hardware Inspection

If available:

- Reseat RAM
- Check SSD/HDD connection
- Listen for beep codes or fan activity

---

## Solutions

| Problem | Fix |
|---------|-----|
| Dead battery | Charge or replace battery |
| Faulty charger | Test with another charger |
| RAM issue | Reseat or replace RAM |
| Hardware failure | Escalate for repair |
