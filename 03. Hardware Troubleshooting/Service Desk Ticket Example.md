# 🎫 Service Desk Ticket Example

## Issue

User reports printer is offline.

## Investigation

Checked printer power, network connection, and print queue.

## Root Cause

Print queue was stuck due to a failed print job.

## Resolution

Cleared the print queue and restarted the printer.

✅ Resolved
