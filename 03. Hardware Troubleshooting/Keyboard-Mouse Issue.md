# ⌨ Keyboard & Mouse Issues

## Issue

Keyboard or mouse is not responding or behaves unexpectedly.

---

## Possible Causes

- Loose USB connection
- Dead batteries (wireless)
- Driver issue
- Hardware failure

---

## Troubleshooting Steps

### 1. Check Connection

Verify:

- USB securely connected
- Bluetooth enabled
- Batteries charged

---

### 2. Test Another USB Port

Move the device to another USB port.

---

### 3. Restart Computer

Simple restart often restores device functionality.

---

### 4. Update Drivers

Open:

```
Device Manager
→ Keyboards
→ Mice and other pointing devices
```

Update or reinstall drivers.

---

## Solutions

| Problem | Fix |
|---------|-----|
| Dead batteries | Replace batteries |
| USB issue | Use another USB port |
| Driver problem | Update/reinstall driver |
| Hardware failure | Replace device |

---

# 🛠 Useful Hardware Tools

| Tool | Purpose |
|------|---------|
| Device Manager | Check hardware status |
| BIOS/UEFI | Detect hardware |
| Windows Memory Diagnostic | Test RAM |
| Task Manager | Monitor hardware usage |
