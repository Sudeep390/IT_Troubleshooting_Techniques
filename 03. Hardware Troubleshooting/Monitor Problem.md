# 🖥 Monitor Problems

## Issue

Monitor shows no display, flickers, or displays poor image quality.

---

## Possible Causes

- Loose cable
- Incorrect input source
- Graphics driver issue
- Faulty monitor
- GPU problem

---

## Troubleshooting Steps

### 1. Check Connections

Verify:

- HDMI/DisplayPort/VGA cable
- Monitor power cable

---

### 2. Check Input Source

Ensure the correct input is selected:

- HDMI
- DisplayPort
- VGA

---

### 3. Test Another Monitor

If available:

- Connect another monitor
- Test using another cable

---

### 4. Update Graphics Driver

Open:

```
Device Manager
→ Display Adapters
```

Update the graphics driver.

---

## Solutions

| Problem | Fix |
|---------|-----|
| Loose cable | Reconnect cable |
| Wrong input | Select correct source |
| Driver issue | Update graphics driver |
| Faulty monitor | Replace monitor |
