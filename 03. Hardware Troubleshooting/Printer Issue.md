# 🖨 Printer Issues

## Issue

Printer is offline, not printing, or print jobs are stuck.

---

## Possible Causes

- Printer offline
- Paper jam
- Low ink/toner
- Driver issue
- Network connection problem

---

## Troubleshooting Steps

### 1. Check Printer Status

Verify:

- Printer powered on
- No error lights
- Paper loaded
- Ink/Toner available

---

### 2. Restart Devices

Restart:

- Printer
- Computer

---

### 3. Check Print Queue

Path:

```
Settings
→ Bluetooth & Devices
→ Printers & Scanners
```

Cancel stuck print jobs.

---

### 4. Verify Connection

Check:

- USB cable
- WiFi connection
- Network cable

---

### 5. Update Printer Driver

Open:

```
Device Manager
```

Update or reinstall the printer driver.

---

## Solutions

| Problem | Fix |
|---------|-----|
| Printer offline | Set printer online |
| Stuck print jobs | Clear print queue |
| Driver issue | Reinstall driver |
| Network issue | Reconnect printer |
